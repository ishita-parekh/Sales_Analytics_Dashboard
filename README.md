# Sales_Analytics_Dashboard
A Sales Analytics Dashboard designed using Power BI to analyze the sales data of Awesome Chocolates sales Case Study. 

Used following functions of PowerBI:
1. Power Query to create new columns and clean the data
2. Created DAX Measures for metrics computation
3. Created interactive dashboard to understand and analyze the metrics based on different base parameters like no of boxes, shipments, costs, sales, and/ or % profit.
